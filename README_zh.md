# 功能主体理论

[English Version](README.md)

## 理论概述

功能主体理论研究：在AI时代，生产、交换、治理与责任关系中的有效主体，如何形成、稳定、演化和退出。

本理论不预设社会运行的基本主体必须始终是：

- 自然人；
- 企业；
- 政府；
- 法律规定的固定组织；
- 其他历史形成的主体类型。

它提出：

> 当一个稳定功能集合能够持续提供外部功能、形成内部决策闭环、协调必要资源、拥有自身的成本收益结构、参与契约或其他协调关系、保存履约证据并承担可识别责任时，这个集合可能形成一个有效的功能主体。

一个功能主体可能由以下部分构成：

- 一个自然人；
- 一个自然人的部分能力；
- 多个自然人；
- 一个或多个AI系统；
- 工具、设备、数据和自动化流程；
- 分布在多个现有组织中的组件；
- 一个稳定的人机混合生产系统。

因此，功能主体的边界主要由以下结构共同决定：

稳定功能
+ 决策闭环
+ 资源协调
+ 交换能力
+ 证据连续性
+ 责任结构

而不只由生物边界、行政身份或法律登记形式决定。

## 核心区分

功能主体理论区分两个不同层级。

自然人作为权利与体验主体

自然人仍然是以下内容的主体：

基本权利；
身体自主；
生活体验；
同意权；
拒绝权；
申诉权；
个体福利；
是否提供某项能力的自主选择权。

## 功能主体作为社会运行主体

功能主体可以成为以下内容的主体：

生产；
交换；
任务承诺；
资源协调；
报价；
治理；
履约证据积累；
运行责任。

这个区分不是取消自然人。

它否定的是：

一个生物个体必须始终对应一个不可拆分的经济主体、组织主体、生产主体和治理主体。

## 能力不等于供给

拥有能力，并不等于自然人或功能主体愿意提供这项能力。

有效供给可能取决于：

预期回报；
直接成本；
机会成本；
时间要求；
风险；
学习收益；
战略价值；
自主程度；
未来收益；
非货币回报。

因此：

能力 ≠ 提供意愿
提供意愿 ≠ 交易成立

只有相关主体之间的可接受契约区间发生重叠，交易或协调关系才可能成立。

价格也不一定只是货币。

它可能是一个多维契约，包括：

货币；
资源使用权；
未来收益权；
风险分配；
时间自由；
学习机会；
声望；
决策权；
长期参与权。

## 功能主体的形成

功能主体并不是通过分配一个角色，或者简单组合若干工具就自动产生。

一个功能集合可能在持续生产中逐步发展为主体，其条件可能包括：

能力互补
+ 持续功能
+ 内部协调优势
+ 决策闭环
+ 资源控制
+ 外部交换能力
+ 证据连续性
+ 责任结构
= 候选功能主体

功能主体的边界是内生形成和持续演化的。

以下变化都可能改变主体边界：

协调成本发生变化；
AI降低搜索、匹配或契约成本；
某些能力变得可复制；
新工具改变生产结构；
责任能够被重新分配；
原有稳定系统失去效率；
临时合作形成持续的功能连续性。

## 与现有机制的关系

功能主体理论本身不是一种单一的资源分配机制或治理机制。

它研究的是更基础的问题：

市场、计划、治理和分配机制，究竟作用于什么样的主体。

功能主体之间可以采用多种协调机制，例如：

市场；
拍卖；
契约；
计划；
平台匹配；
社区自治；
多中心治理；
公共预算；
协商；
混合机制。

理论层级可以表示为：

主体本体
→ 价值与价格形成
→ 分配与协调机制
→ 治理、审计与纠错

市场理论、机制设计、能力方法、交易成本经济学和多中心治理，都可以被纳入这个理论框架，而不需要被它取代。

## 当前研究问题

本项目当前重点研究以下问题：

一个功能集合在什么条件下成为独立主体，而不只是一个组件？
功能主体的边界应当如何识别？
内部组件之间如何分配收益、成本和责任？
功能主体如何形成自己的偏好、成本和保留价格结构？
功能主体如何形成、合并、分裂、演化、复制和退出？
当生产主体越来越非生物化时，如何保护自然人的基本权利？
如何防止功能主体形成永久权力、垄断和自我保存结构？
人、AI、组织和技术组件之间的法律责任应当如何穿透？
当前哪些组织已经在向功能主体演化？
在什么条件下，功能主体比传统企业和机构更有效或更具正当性？
